# API WEATHER

> Support for POST and GET requests


#### URL: http://localhost:{{PORT}} or http://localhost:3004

| Request | Endpoint     |
|---------|--------------|
| GET     | /listWeather |
| POST    | /addWeather  |

Content-type: x-www-form-urlencoded

Parameters required on body:
| Key         | value |
|-------------|-------|
| humidity    |  int  |
| temperature |  int  |
